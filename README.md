# Bakalavr
Бакалаврська кваліфікаційна робота Мельничука Юліана РІ-42 з РІС

# Розгортання роботи
Достатньо запулити цей гіт-репозиторій та написати наступну команду в cmd у директорії даного проєкту:
```
docker compose up -d --build
```
